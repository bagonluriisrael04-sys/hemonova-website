# HemoNova HealthTech

HemoNova HealthTech is a medical technology startup focused on developing innovative solutions for **non-invasive anemia detection**. The goal of the project is to provide a faster, safer, and pain-free method of screening hemoglobin levels without the need for blood samples.

## About the Project

This website serves as the official landing page for the HemoNova HealthTech concept. It presents the problem of global anemia, introduces the technology approach, and outlines the startup’s roadmap and projected impact.

Anemia affects more than **1.6 billion people worldwide**, especially pregnant women, children, and vulnerable populations. Traditional testing methods require blood draws and laboratory equipment, which can limit access to early screening.

HemoNova HealthTech aims to improve accessibility to anemia detection through **portable, rapid, and non-invasive technology**.
